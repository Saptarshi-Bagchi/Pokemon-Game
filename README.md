# ⚔️ Pokémon Battle Simulator

A command-line Pokémon battle game built in Python that simulates turn-based combat with type effectiveness, status conditions, and unique moves for each Pokémon class.

## 🚀 Features
- Turn-based battle system  
- Type effectiveness (Fire, Water, Grass, Electric)  
- Status effects (e.g., paralysis)  
- Unique moves with damage, recoil, and healing  
- Simple CLI interaction  

## 📦 Requirements
- Python 3.x

## ▶️ How to Run
```bash
python pokemon_official.py
